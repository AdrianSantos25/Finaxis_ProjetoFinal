# Comprovativos de Transações

Esta pasta armazena os arquivos de comprovativos (recibos, notas fiscais, etc) enviados pelos utilizadores nas suas transações.

## Formatos Aceites
- Imagens: JPEG, PNG
- Documentos: PDF

## Limite de Tamanho
- Máximo: 5MB por arquivo

## Segurança
Os arquivos são armazenados com nomes únicos para evitar conflitos e são acessíveis apenas através da aplicação autenticada.
